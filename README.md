# Light Path Puzzle

A C# console-based puzzle game where players rotate mirrors to guide a light beam from the source to the target while avoiding obstacles.

## Features

* Interactive console gameplay
* Mirror rotation mechanics
* Light reflection simulation
* Win/lose detection

## Technologies

* C#
* .NET

## How to Run

1. Open the project in Visual Studio.
2. Build and run the application.
3. Rotate mirrors or emit the light beam.
4. Guide the light to the target to win.
